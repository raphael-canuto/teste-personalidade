# Teste de Personalidade

Este projeto foi desenvolvido como atividade da disciplina de Algoritmos e consiste em um teste de personalidade interativo. O usuário informa seus níveis em diferentes características e, a partir de estruturas condicionais implementadas em JavaScript, o sistema analisa os dados e retorna qual personagem ele mais se assemelha.
